# Student Mental Health & Academic Impact - ML vs. DL Comparative Pipeline

This repository contains the codebase and technical execution for the Summative Assignment on Model Training and Evaluation. The project develops a comprehensive benchmark pipeline evaluating traditional Machine Learning (Scikit-learn, XGBoost) against Deep Learning paradigms (TensorFlow Sequential and Functional APIs) to classify academic impact levels.

## 🚀 Project Overview & Mission Alignment
- **Problem Statement:** Analyzing how multi-dimensional student profiles (academic pressure, anxiety levels, sleep hours) correlate with academic performance and mental health indicators.
- **Dataset Source:** Obtained from open-source repositories from kaggle.
Kaggle Link: https://www.kaggle.com/datasets/sehaj1104/student-mental-health-and-burnout-dataset
- **Core Methodology:** An end-to-end engineered pipeline utilizing zero-data-leakage preprocessing, automated model tracking, hyperparameter exploration across 7 distinct experiments, and structural feature importance extraction.

## 📊 Repository Architecture & Deliverables
- `summative_training.ipynb`: Main Google Colab/Jupyter notebook containing fully modular, executable code with inline visualizations (Learning Curves, Confusion Matrices, ROC/AUC Curves).
- `requirements.txt`: Specified library dependencies ensuring absolute compute environment replication.
- `README.md`: Comprehensive project overview and documentation.
